# Testing

## Test Profiles

Every pipeline has a `test` profile in nextflow.config:
```groovy
test {
    params.samplesheet = "${projectDir}/test_data/samplesheet_test.csv"
    params.outdir      = "${projectDir}/results_test"
    params.max_memory   = '8.GB'
    params.max_cpus     = 4
    params.max_time     = '2.h'

    docker { enabled = true }
    process { cpus = 4; memory = 8.GB; time = 2.h }
}
```

## Running Tests

```bash
# Stub test (validates workflow wiring without real data)
nextflow run main.nf -profile test -stub

# Real test with test data
nextflow run main.nf -profile test

# With pixi
pixi run nextflow run main.nf -profile test
```

## Module Test Template (nf-test)

Location: `modules/local/PROCESS_NAME/tests/main.nf.test`

```groovy
nextflow_process {
    name "Test Process PROCESS_NAME"
    script "../main.nf"
    process "PROCESS_NAME"
    tag "modules"
    tag "modules_local"

    test("Should process sample successfully") {
        when {
            process {
                """
                input[0] = tuple(
                    'test_sample',
                    file("${projectDir}/test_data/input.txt")
                )
                """
            }
        }
        then {
            assertAll(
                { assert process.success },
                { assert process.out.result.size() == 1 },
                { assert process.out.result.get(0).get(1).exists() }
            )
        }
    }

    test("Should process sample - stub") {
        options "-stub"
        when {
            process {
                """
                input[0] = tuple('test', file("dummy.txt"))
                """
            }
        }
        then {
            assertAll(
                { assert process.success },
                { assert snapshot(process.out).match() }
            )
        }
    }
}
```

## Running Module Tests

```bash
nf-test test modules/local/process_one/tests/main.nf.test
nf-test test --tag modules_local
nf-test test --update-snapshot
```

## Test Data Sources
- Create minimal subsamples with `seqtk sample`
- Use nf-core test datasets: https://github.com/nf-core/test-datasets
- Extract small regions with `samtools view -b chr22`

## Testing Checklist

Before production:
- [ ] Stub test passes (`-stub`)
- [ ] Local test passes with test data (`-profile test`)
- [ ] versions.yml generated by main processes
- [ ] publishDir outputs are correct
- [ ] Test on GCP with small dataset
- [ ] Execution reports reviewed
